# 概念



## 鉴权(Authentication)

通俗的讲，就是验证当前用户身份，证明“你是你自己”。

## 授权(Authorization)

用户授予第三方应用访问该用户某些资源的权限。例如，你在安装应用的时候，APP会询问是否允许授予权限（通讯录、地理位置等）。

## 凭证(Credentials)

实现鉴权和授权的前提是需要一种媒介（证书）来标记访问者的身份。例如我们生活中的身份证。
